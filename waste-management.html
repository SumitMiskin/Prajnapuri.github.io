Waste Management (Collection & Recycling)

The city operates an automated waste collection system where trash bins report fill-levels to optimize garbage truck routes and ensure efficient resource recovery.

Core Data Structures
  • Arrays / Structs: Store each bin sensor’s data (ID, location, fill percentage).
  • Graphs: Represent collection routes where nodes are bins/depots and edges are roads.
  • Adjacency Lists: Storage format for the route graph to enable efficient traversal.
  • Union-Find: Groups nearby bins into clusters for localized collection (bins in the same neighborhood form a set).
  • Tries / Hash Maps: used to index bin IDs or addresses for rapid lookup.
Routing & Graph Algorithms
  • Dijkstra’s Algorithm: Computes shortest paths between bins (using a priority queue) to minimize travel distance/fuel.
  • Bellman-Ford: Can handle negative edge weights, useful if modeling priority pickups as "negative cost" in route planning.
  • Kruskal’s MST: Used to connect new recycling centers to the grid cost-effectively (Minimum Spanning Tree).
  • BFS / DFS: Breadth-First Search sweeps sectors (e.g., finding nearest full bins), while DFS ensures all bins are reachable.
  • Floyd-Warshall: Pre-computes all-pairs shortest paths to evaluate complex route combinations.
Range and Dynamic Queries
  • Segment Tree: Enables O(log n) queries for total waste statistics in a specific range of city blocks.
  • Fenwick Tree (BIT): Maintains running totals for continuous sensor data streams (e.g., total kg waste collected) with O(log n) updates.
  • Sparse Table: used on historical data to answer static queries like "max fill-level last week" in O(1) time.
Sorting & Scheduling
  • Max-Heap (Priority Queue) / Heap Sort: Selects the fullest or highest-priority bins for immediate collection.
  • Selection / Bubble Sort: Efficient enough for sorting small batches (e.g., < 50 bins) during local route adjustments.
  • Skip List: Manages a dynamic, sorted list of upcoming pickups with O(log n) average insertion time.
Pattern Search
  • String Algorithms (Boyer-Moore, Rabin-Karp, KMP): Rapidly scan large text streams from social media or logs to detect complaints (e.g., "overflowing garbage").
  • Brute-Force Search: applied for simple keyword matching in short messages.
