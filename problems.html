Shortest Path Navigation
Problem Statement

Find the quickest path in a weighted city graph where edge weights reflect traffic delays and time cost.

Solution Approach

Use Dijkstra with a priority queue; edge weights are non-negative, complexity O(E log V).

C++ Implementation
#include <bits/stdc++.h>
using namespace std;
void dijkstra(int src, vector>>& g){
  const int INF=1e9; int n=g.size();
  vector dist(n,INF); dist[src]=0;
  priority_queue, vector>, greater>>pq;
  pq.push({0,src});
  while(!pq.empty()){
    auto [d,u]=pq.top(); pq.pop();
    if(d!=dist[u]) continue;
    for(auto &e: g[u]){
      int v=e.first,w=e.second;
      if(dist[v]>d+w){ dist[v]=d+w; pq.push({dist[v],v}); }
    }
  }
}

Verify reachability of stations from a central hub using BFS traversal.

Implementation
// BFS traversal example
void bfs(int s, vector>& g){
  vector vis(g.size()); queueq; q.push(s); vis[s]=1;
  while(!q.empty()){ int u=q.front(); q.pop(); for(int v:g[u]) if(!vis[v]){ vis[v]=1; q.push(v);} }
}
