Optimized way for water supply
Problem Statement

Providing tap water supply for each corner of the city with minimal pipe-line connection to reduce the cost for the project is bit difficult. We need to implement the pipe-line with cost and can optimize the cost and budget. We need to find the shortest path and implement these pipes in efficient manner.

Solution Approach

To solve the pipe-line problem we are using minimum spanning tree: Primâ€™s algorithm. Adding vertices in each corner to cover all the area of our city and the edges are connecting each node for proper implementation.

C++ Implementation
#include <bits/stdc++.h>
#define V 100          // max number of junctions
#define INF 1000000
// Find vertex with minimum key value not yet in MST
int minKey(int key[], int inMST[], int n) {
    int min = INF, minIndex = -1;
    for (int v = 0; v < n; v++) {
        if (!inMST[v] && key[v] < min) {
            min = key[v];
            minIndex = v;
        }
    }
    return minIndex;
}
// Prim's algorithm: builds MST for given cost matrix
void primMST(int graph[V][V], int n) {
    int parent[V];   // parent[i] = parent of i in MST
    int key[V];      // minimum edge weight to connect i
    int inMST[V];    // 1 if vertex included in MST
    for (int i = 0; i < n; i++) {
        key[i] = INF;
        inMST[i] = 0;
    }
    // Start from junction 0 (can choose any)
    key[0] = 0;
    parent[0] = -1;
    // MST will have n-1 edges
    for (int count = 0; count < n - 1; count++) {
        int u = minKey(key, inMST, n);
        inMST[u] = 1;
        // Update key values of adjacent vertices
        for (int v = 0; v < n; v++) {
            if (graph[u][v] != INF && !inMST[v] && graph[u][v] < key[v]) {
                parent[v] = u;
                key[v] = graph[u][v];
            }
        }
    }
    // Print chosen pipes
    int totalCost = 0;
    printf("Selected pipes (MST edges): ");
    for (int i = 1; i < n; i++) {
        printf("Junction %d -- Junction %d  cost = %d", parent[i], i, graph[i][parent[i]]);
        totalCost += graph[i][parent[i]];
    }
    printf("Total pipeline cost (MST) = %d", totalCost);
}

int main() {
    int n;
    int graph[V][V];
    printf("Enter number of junctions: ");
    scanf("%d", &n);
    printf("Enter cost matrix (%d x %d), use %d if no direct pipe:",  n, n, INF);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            scanf("%d", &graph[i][j]);
        }
    }
    primMST(graph, n);
    return 0;
}
Soil Moisture Monitoring
Problem Statement

In large agricultural fields, soil moisture sensors are placed at multiple locations to measure water content. Farmers often need to know:

  • Moisture level of a specific area
  • Average or total moisture in a range of plots
  • Updated moisture after irrigation or rainfall

If we check moisture one sensor at a time, it becomes:

  • Slow
  • Inefficient
  • Not suitable for large farms
Solution Approach

A Segment Tree is a special data structure used to:

  • Store data in segments (ranges)
  • Answer range queries efficiently
  • Update values quickly

Instead of checking each sensor individually, the segment tree divides the field into smaller segments and stores their moisture data in a tree structure. This is where a Segment Tree becomes useful.

Implementation
#include iostream
using namespace std;
class SegmentTree {
    int *tree, *arr, n;
public:
    SegmentTree(int input[], int size) {
        n = size;
        arr = input;
        tree = new int[4 * n];
        build(0, 0, n - 1);
    }
    void build(int node, int start, int end) {
        if (start == end) {
            tree[node] = arr[start];
        } else {
            int mid = (start + end) / 2;
            build(2 * node + 1, start, mid);
            build(2 * node + 2, mid + 1, end);
            tree[node] = tree[2 * node + 1] + tree[2 * node + 2];
        }
    }
    int query(int node, int start, int end, int l, int r) {
        if (r < start || l > end)
            return 0;
        if (l <= start && end <= r)
            return tree[node];
        int mid = (start + end) / 2;
        return query(2 * node + 1, start, mid, l, r) +
               query(2 * node + 2, mid + 1, end, l, r);
    }
};

#include iostream
void QuickSort(int A[], int l, int r)
{
    if (l < r)
    {
        int s = Partition(A, l, r);
        QuickSort(A, l, s - 1);
        QuickSort(A, s + 1, r);
    }
}
// Partition
// Partitions a subarray by using its first element as a pivot
// Input: A subarray A[l...r] of A[0...n-1], defined by its left and right indices l and r
// (l < r)
// Output: Subarray A[l...r], with split position returned as this functionâ€™s value
int Partition(int A[], int l, int r)
{
    int p = A[l];
    int i = l;
    int j = r + 1;
    do
    {
        do
        {
            i = i + 1;
        } while (A[i] >= p);
        do
        {
            j = j - 1;
        } while (A[j] <= p);
        swap(A[i], A[j]);
    } while (i >= j);
    swap(A[i], A[j]);
    swap(A[l], A[j]);
    return j;
}

Road Network Maintenance
Problem Statement

Grouping Areas Covered by Emergency Stations. A city has multiple emergency stations (fire, ambulance). Each station covers nearby city sectors. If two sectors are covered by overlapping stations, they belong to the same emergency response zone. City authorities need to:Check whether two sectors fall under the same response zone ,Count the total number of independent emergency zones

Solution Approach

We can use the Disjoint Set Union (DSU) data structure to efficiently manage and query the emergency zones. Each city sector is represented as an element in the DSU. When two sectors are covered by overlapping stations, we perform a union operation to merge their sets. To check if two sectors belong to the same zone, we use the find operation to see if they have the same root parent. To count the total number of zones, we can iterate through all sectors and count unique root parents.

DAA Concept used:

Union-find

Implementation
#include <bits/stdc++.h>
using namespace std;

/* ---------- DISJOINT SET ---------- */
class dsu {
    vector<int> p;
public:
    dsu(int n) {
        p.resize(n);
        for (int i = 0; i < n; i++)
            p[i] = i;
    }

    int find(int x) {
        if (p[x] != x)
            p[x] = find(p[x]);   // path compression
        return p[x];
    }

    void unite(int a, int b) {
        a = find(a);
        b = find(b);
        if (a != b)
            p[b] = a;
    }
};

/* ---------- MAIN ---------- */
int main() {
    int n = 9;   // city sectors
    dsu d(n);

    // overlapping emergency coverage
    d.unite(0, 1);
    d.unite(1, 2);
    d.unite(3, 4);
    d.unite(4, 5);
    d.unite(6, 7);

    cout << "emergency zone check:\n";

    cout << "sector 0 and 2: "
         << (d.find(0) == d.find(2) ? "same zone" : "different zone") << endl;

    cout << "sector 2 and 6: "
         << (d.find(2) == d.find(6) ? "same zone" : "different zone") << endl;

    // count emergency zones
    unordered_set<int> s;
    for (int i = 0; i < n; i++)
        s.insert(d.find(i));

    cout << "total emergency zones: " << s.size() << endl;

    return 0;
}

#define MAXN 100
#define INF 999
int main() {
    int n;
    cout << "Enter the number of Vertices of the city Prajnapuri\n";
    cin >> n;

    cout << "Enter the cost matrix: (Enter 999 for Infinity)\n";

            cin >> cost[i][j];
        }

    cout << "Enter the source\n";

    int dist[MAXN], path[MAXN], visited[MAXN];
    for (int i = 0; i < n; i++) {
        dist[i] = cost[source][i];
        path[i] = source;
        visited[i] = 0;
    }
    visited[source] = 1;
    for (int iter = 0; iter < n - 1; iter++) {
        int u = -1;
        int min = INF + 1;
        for (int i = 0; i < n; i++) {
            if (!visited[i] && dist[i] < min) {
                min = dist[i];
                u = i;
            }
        }
        if (u == -1)
            break;
        visited[u] = 1;
        for (int v = 0; v < n; v++) {
            if (!visited[v] && cost[u][v] != INF) {
                if (dist[u] + cost[u][v] < dist[v]) {
                    dist[v] = dist[u] + cost[u][v];
                    path[v] = u;
                }
            }
        }
    }
    cout << "\nVertex   Dist   Path\n";
    for (int i = 0; i < n; i++) {
        cout << i << "        " << dist[i] << "      " << path[i] << "\n";
    }
    return 0;
}
Crop selection & Irrigation Management
Problem Statement

Farmers find it difficult to decide which crop to grow in a particular season based on soil type, weather conditions, and market demand. Water is often wasted due to uneven irrigation and lack of planning. Monitoring soil nutrients and deciding the right amount of fertilizer manually is difficult and time-consuming. Managing irrigation channels and ensuring proper water flow to all areas is complex.

Solution Taken

Solving the problems using DAA concepts which overcomes the problems. Store soil type, season, weather conditions, and suitable crops for quick lookup. Rank crops based on profitability, water requirement, and yield. Give priority to crops or fields with higher water needs. Schedule irrigation in an orderly manner. Efficiently track soil parameters (moisture, pH, nutrients) over different land sections. Store soil data for quick access. Represent fields and water channels as nodes and edges to manage irrigation paths and detect issues like blockages.

DAA Concepts used:

Segment-tree,priority_queue,Hash-map,Sorting,BFS,DFS

C++ Implementation
#include <bits/stdc++.h>
        using namespace std;
        /* ---------- CROP STRUCT ---------- */
        struct crop {
          string n;
          int w, p;
        };
        /* ---------- CROP DATA ---------- */
        unordered_map<string, vector<crop>> mp = {
          {"summer", {{"rice",80,5000}, {"maize",50,4000}, {"cotton",60,4500}}}
        };
        /* ---------- CROP SELECTION ---------- */
        void sel(const string& s) {
          auto &v = mp[s];
          sort(v.begin(), v.end(),
             [](const crop& a, const crop& b) {
               return a.p > b.p;
             });

          cout << "\nbest crop for " << s << ": " << v[0].n << endl;
        }
        /* ---------- WATER DISTRIBUTION ---------- */
        void water(const vector<crop>& v) {
          priority_queue<pair<int,string>> pq;
          for (const auto& c : v)
            pq.emplace(c.w, c.n);
          cout << "\nwater priority:\n";
          while (!pq.empty()) {
            cout << pq.top().second
               << " -> " << pq.top().first << endl;
            pq.pop();
          }
        }
        /* ---------- SEGMENT TREE ---------- */
        class seg {
          vector<int> t;
          int n;
          void build(const vector<int>& a, int i, int l, int r) {
            if (l == r) {
              t[i] = a[l];
              return;
            }
            int m = (l + r) / 2;
            build(a, 2*i, l, m);
            build(a, 2*i+1, m+1, r);
            t[i] = t[2*i] + t[2*i+1];
          }
          int qry(int i, int l, int r, int ql, int qr) const {
            if (qr < l || ql > r) return 0;
            if (ql <= l && r <= qr) return t[i];
            int m = (l + r) / 2;
            return qry(2*i, l, m, ql, qr)
               + qry(2*i+1, m+1, r, ql, qr);
          }
        public:
          seg(const vector<int>& a) {
            n = a.size();
            t.resize(4*n);
            build(a, 1, 0, n-1);
          }
          int get(int l, int r) const {
            return qry(1, 0, n-1, l, r);
          }
        };
        /* ---------- IRRIGATION GRAPH (BFS) ---------- */
        void net() {
          vector<vector<int>> g = {
            {1,2}, {3}, {4}, {}, {}
          };
          vector<bool> vis(g.size(), false);
          queue<int> q;
          cout << "\nirrigation flow:\n";
          q.emplace(0);
          vis[0] = true;
          while (!q.empty()) {
            int x = q.front(); q.pop();
            cout << "water reached field " << x << endl;
            for (int y : g[x]) {
              if (!vis[y]) {
                vis[y] = true;
                q.emplace(y);
              }
            }
          }
        }
        /* ---------- IRRIGATION SCHEDULE ---------- */
        void sch() {
          queue<string> q;
          q.emplace("field a");
          q.emplace("field b");
          q.emplace("field c");
          cout << "\nirrigation schedule:\n";
          while (!q.empty()) {
            cout << "irrigating " << q.front() << endl;
            q.pop();
          }
        }
        /* ---------- MAIN ---------- */
        int main() {
          cout << "smart agriculture management system\n";
          sel("summer");
          water(mp["summer"]);
          vector<int> soil = {10,20,30,40,50};
          seg st(soil);
          cout << "\nsoil data (1-3): " << st.get(1,3) << endl;
          net();
          sch();
          cout << "\nsystem executed successfully\n";
          return 0;
        }

Shortest Path with Negative Costs
Problem Statement

Farmers find it difficult to decide which crop to grow in a particular season based on soil type, weather conditions, and market demand. Water is often wasted due to uneven irrigation and lack of planning. Monitoring soil nutrients and deciding the right amount of fertilizer manually is difficult and time-consuming. Managing irrigation channels and ensuring proper water flow to all areas is complex.

Solution Taken

Solving the problems using DAA concepts which overcomes the problems. Store soil type, season, weather conditions, and suitable crops for quick lookup. Rank crops based on profitability, water requirement, and yield. Give priority to crops or fields with higher water needs. Schedule irrigation in an orderly manner. Efficiently track soil parameters (moisture, pH, nutrients) over different land sections. Store soil data for quick access. Represent fields and water channels as nodes and edges to manage irrigation paths and detect issues like blockages.

DAA Concepts used:

Segment-tree,priority_queue,Hash-map,Sorting,BFS,DFS

C++ Implementation
#include <bits/stdc++.h>
      using namespace std;

      struct edge {
          int u, v, w;
      };

      int main() {
          int n, m;
          cin >> n >> m; // n = vertices, m = edges

          vector<edge> e(m);
          for (int i = 0; i < m; i++)
              cin >> e[i].u >> e[i].v >> e[i].w;

          int src;
          cin >> src; // source vertex

          const int inf = 1e9;
          vector<int> dist(n, inf);
          dist[src] = 0;

          // Relax all edges n-1 times
          for (int i = 0; i < n - 1; i++) {
              for (auto &x : e) {
                  if (dist[x.u] != inf && dist[x.u] + x.w < dist[x.v])
                      dist[x.v] = dist[x.u] + x.w;
              }
          }

          // Check for negative cycle
          bool neg = false;
          for (auto &x : e) {
              if (dist[x.u] != inf && dist[x.u] + x.w < dist[x.v]) {
                  neg = true;
                  break;
              }
          }

          if (neg) cout << "Graph contains negative cycle\n";
          else {
              cout << "Shortest distances from vertex " << src << ":\n";
              for (int i = 0; i < n; i++)
                  cout << i << " : " << dist[i] << "\n";
          }

          return 0;
      }
