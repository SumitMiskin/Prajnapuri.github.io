Problem Statement

Garbage trucks need a flexible route system where stops can be added, removed, or reordered dynamically based on bin fill levels. A linear list isn't efficient for bi-directional traversal (e.g., driver backtracking).

Solution Approach

We use a **Doubly Linked List (DLL)**. Each node represents a waste bin location. The `next` and `prev` pointers allow the truck driver to navigate both forward and backward along the route. Insertions and deletions at any point are efficient, making it perfect for dynamic route adjustments.
