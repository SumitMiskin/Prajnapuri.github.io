Problem Statement

Managing a doctor's daily appointment list requires sorting patients by their scheduled time to ensure a smooth workflow. Manual sorting can be error-prone and inefficient for administrative staff.

Solution Approach

We implement **Selection Sort**, a simple and intuitive sorting algorithm. It repeatedly selects the earliest remaining appointment time and moves it to the correct position (O(n²)). Checks for the smallest element in the unsorted portion and swaps it with the first unsorted element. While not the fastest for massive datasets, it is efficient enough and easy to implement for small daily schedules.

C++ Implementation
1#include <iostream>

4


5struct Appointment {


6    string patientName;


7    int appointmentTime; // in 24hr format, e.g., 930 = 9:30 AM


8};


9


10void selectionSort(Appointment arr[], int n) {


11    for (int i = 0; i < n - 1; i++) {


12        int minIdx = i;


13        for (int j = i + 1; j < n; j++) {


14            if (arr[j].appointmentTime < arr[minIdx].appointmentTime) {


15                minIdx = j;


16            }


17        }


18        swap(arr[i], arr[minIdx]);


19    }


20}
