Problem Statement

Efficiently tracking waste collection data from various locations is essential for municipal waste management. We need a structured way to store and retrieve details like location, waste type, and weight for analysis.

Solution Approach

We use a **Structure (struct)** to define a complex data type `WasteRecord` that groups related information (location, type, weight). An **Array** of these structures enables us to manage multiple records systematically, allowing for easy data entry and reporting.
