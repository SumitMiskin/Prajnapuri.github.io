Course Reflection

Kinds of Problems in Nature

We can see many examples in nature that are similar to concepts like iteration, recursion, and backtracking. The change in seasons happens in a cycle again and again, which is like iteration. The growth of cabbage leaves in layers is like recursion, where the same pattern repeats itself. Ants exploring their area and returning home is like backtracking, as they explore paths and then come back. There are many other examples in nature that match these ideas.

Space and Time Efficiency

Space complexity is the extra memory an algorithm needs to run. Time complexity is the time an algorithm takes to complete its task. Both are important to check how well an algorithm works and if it fits a certain problem. The order of growth shows how the time or space needed grows with the input size:

  • O (1) → constant growth
  • O (n) → linear growth
  • O (log n) → logarithmic growth

Takeaway from Design Principles: After learning different algorithms, the main ideas became clear. AVL trees and Red-Black trees taught how balance is important for fast operations. The comparison between DFS (Depth First Search) and BFS (Breadth First Search) showed how different methods can be used to solve problems. Learning how different sorting algorithms work—like dividing and merging or swapping elements—helped in understanding their working logic. Altogether, these ideas gave a good base for understanding the principles of algorithms.

Hierarchical Data and Tree Data Structures

Tree data structures help in organizing data in a hierarchy and solve many problems efficiently. Binary Search Trees (BSTs) help in searching faster but are not always balanced. A 2-3 tree keeps all leaf nodes at the same level, making it more balanced but still not perfect. An AVL tree is balanced using a balancing factor and rotations. The Red-Black tree improves this by reducing the number of rotations. A heap is a complete binary tree used to maintain a heap property, good for priority queues. A trie is used to store strings, useful in autocomplete and prefix search systems.

Need for Array Query

Algorithms array queries are useful for quickly handling large data with frequent changes. Lookup tables store precomputed results for faster queries. Segment trees help perform fast range queries and updates. Binary Indexed Trees (BIT) offer simpler code for range queries and work using the last bit concept.

Difference Between Tree and Graphs

Trees and graphs both store data, but they are different. A tree has one root node and no cycles. A graph has nodes connected by edges and can have cycles.

Tree traversals:

  • Pre-order
  • In-order
  • Post-order

Graph traversals:

  • DFS (Depth First Search) → Brave traversal
  • BFS (Breadth First Search) → Cautious traversal

Trees are used for hierarchical data, sorting, and searching. Graphs are used in problems like finding the shortest path.

Sorting and Searching Algorithms

Sorting and searching make it easy to manage and find information. Sorting, like quick sort and merge sort, arranges data in order. For example, arranging products on shopping websites. Searching, like binary search, helps find specific data quickly, such as search suggestions or product searches online.

Importance of Graph Algorithms

Graph algorithms are important for finding minimum spanning trees (MST) and shortest paths. The MST connects all nodes with the least total cost, useful for laying roads or pipelines. Shortest path algorithms like Dijkstra's help find the fastest route, useful in GPS navigation systems.

Algorithm Design Techniques

Different design methods of algorithms include:

  1. Backtracking – Trying all options and removing wrong ones (like the N-Queens problem).
  2. Sorting algorithms:
    • Divide and conquer: Quick sort and Merge sort
    • Comparison-based: Selection, Insertion, and Bubble sort
  3. Brute force – Trying every possible solution one by one.
  4. String search algorithms:
    • Boyer-Moore
    • Knuth-Morris-Pratt
    • Rabin-Karp
  5. Shortest path algorithms:
    • Dijkstra's
    • Floyd-Warshall
    • Prim's
