Problem Statement

In a busy hospital, quickly retrieving patient records by ID is crucial for emergency care and checking medical history. The database must be kept sorted to enable fast lookups.

Solution Approach

We use **Bubble Sort** (for simplicity in small daily lists) to organize patient IDs and **Binary Search** to locate a specific patient in **O(log n)** time. This ensures that even with a growing list of daily admissions, staff can verify a patient's presence in the system almost instantly.

4// Binary search to locate a patient ID in sorted records


5int binarySearch(int patients[], int size, int targetID) {


6    int low = 0;


7    int high = size - 1;


8


9    while (low <= high) {


10        int mid = low + (high - low) / 2;


11


12        if (patients[mid] == targetID) {


13            return mid;


14        }


15        if (patients[mid] < targetID) {


16            low = mid + 1;


17        } else {


18            high = mid - 1;


19        }


20    }


21    return -1;


22}

24// Bubble sort to sort patient IDs


25void bubbleSort(int patients[], int size) {


26    for (int i = 0; i < size - 1; i++) {


27        for (int j = 0; j < size - i - 1; j++) {


28            if (patients[j] > patients[j + 1]) {


29                int temp = patients[j];


30                patients[j] = patients[j + 1];


31                patients[j + 1] = temp;


32            }


33        }


34    }


35}
