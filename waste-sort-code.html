Problem Statement

To optimize waste collection routes, it is often necessary to prioritize bins that are heavier or fuller. We need a way to sort waste bins based on their weight to identify which ones require immediate attention.

Solution Approach

We use **Bubble Sort**, a simple sorting algorithm that repeatedly steps through the list of bin weights, compares adjacent elements, and swaps them if they are in the wrong order. This sorts the bins by weight in ascending order, helping to prioritize collection tasks.

4void bubble_sort(int n, float *weights) {


5    float temp = 0;


6    for (int i = 0; i <= n - 2; i++) {


7        for (int j = 0; j <= n - 2 - i; j++) {


8            if (weights[j + 1] < weights[j]) {


9                temp = weights[j];


10                weights[j] = weights[j + 1];


11                weights[j + 1] = temp;


12            }


13        }


14    }


15}
